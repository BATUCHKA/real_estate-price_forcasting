# Дараагийн алхмууд

## 1. Бүрэн өгөгдөл цуглуулах
```bash
python scrape_unegui.py --pages 50 --out data/listings.csv
```
~3000 зар, ~2 цаг. Шөнө ажиллуулахад тохиромжтой.

## 2. Өгөгдөл цэвэрлэх (`02_clean.ipynb`)
- `price_mnt` дутуу мөрийг хасах
- Outlier шалгах: үнэ < 10сая ₮ эсвэл > 5тэрбум ₮ → түрээсийн зар байж магадгүй
- `area_m2` < 15 эсвэл > 500 → эргэлзээтэй
- `year_built` < 1950 эсвэл > 2027 → алдаа
- Дүүрэг тус бүрийн зарын тоог шалгах (≥30 байх ёстой)

## 3. EDA (`03_eda.ipynb`)
- Үнэ vs талбай scatter plot
- Дүүрэг тус бүрийн дундаж м²-ын үнэ (boxplot)
- Корреляцийн матриц
- Үнийн log-transformation шалгах (right-skewed байх магадлалтай)

## 4. Загварчлал (`04_models.ipynb`)
Рубрикт **2 оноо** авахын тулд **3+ загвар** харьцуулах:

| Загвар | Зорилго |
|---|---|
| **Шугаман регресс** | Baseline, коэффициент тайлбар |
| **Lasso/Ridge** | Регулярчилал, чухал хувьсагч сонгох |
| **Шийдвэрийн мод** | Шугаман бус хамаарал, тайлбарлахад хялбар |
| **Random Forest** | Хамгийн өндөр нарийвчлал |

**Үнэлгээний хэмжүүр**: RMSE, MAE, R² (тест дээр)

**Тайлбарлах**: коэффициент / feature_importance / SHAP

## 5. Тайлан (Quarto — рубрикт +1 оноо)

```bash
quarto create-project report --type book
```

Бүлгүүд:
1. Хураангуй (1 хуудас)
2. Оршил — асуудал, зорилго
3. Өгөгдөл — эх сурвалж, цуглуулсан арга, цэвэрлэлт
4. Арга зүй — загвар бүрийн томьёо, гипотез
5. Үр дүн — харьцуулсан хүснэгт, графикууд
6. Дүгнэлт — гол олигуудын үнэ хэрхэн тогтдог
7. Ашигласан материал (APA)
8. Хавсралт — код
