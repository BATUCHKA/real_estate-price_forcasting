# Үл хөдлөх хөрөнгийн үнийн таамаглал

НУМ-ийн "Магадлал" хичээлийн багийн төслийн ажил. unegui.mn-ээс цуглуулсан орон сууцны зарын өгөгдөл дээр машин сургалтын регрессийн загвараар үнэ таамаглах.

## Сэдэв ба зорилго

**Асуудал**: Орон сууцны үнэ нь талбай, байршил, барилгын нас, давхарлалт, тоноглол зэрэг олон хүчин зүйлээс хамаардаг. Эдгээр хүчин зүйлийг ашиглан үнийг хэр нарийвчилж урьдчилан таамаглаж болох вэ?

**Зорилго**:
1. Бодит зах зээлийн өгөгдөл цуглуулах (web scraping)
2. Хувьсагчдын хамаарлыг статистик аргаар судлах
3. Регрессийн хэд хэдэн загвар харьцуулах (Шугаман, Lasso, Шийдвэрийн мод, Random Forest)
4. Үнэд хамгийн их нөлөөлөх хүчин зүйлсийг тогтоох

## Ажиллуулах дараалал

### 1. Өгөгдөл цуглуулах

```bash
# Туршилт (~2 минут, 60 зар)
python scrape_unegui.py --pages 1 --out data/test.csv

# Бүрэн (~2 цаг, ~3000 зар) — шөнө ажиллуулах нь зүйтэй
python scrape_unegui.py --pages 50 --out data/listings.csv
```

Скрипт нь өмнө татсан зарыг алгасдаг тул дахин ажиллуулахад аюулгүй.

### 2. Цэвэрлэлт

```bash
jupyter notebook notebooks/02_clean.ipynb
```

`data/listings.csv` → `data/listings_clean.csv`

Хийгддэг үйлдлүүд:
- Давхардсан зар арилгах
- Үнэ, талбай, оны outlier хасах
- Категорийн хувьсагч кодлох (`has_balcony`, `has_garage`, `has_elevator`)
- Шинэ feature: `price_per_m2`, `building_age`, `log_price`, `floor_ratio`

### 3. EDA

```bash
jupyter notebook notebooks/03_eda.ipynb
```

Үнийн тархалт, корреляцийн матриц, дүүрэг тус бүрийн боксплот, гипотез тогтоох.

### 4. Загварчлал

```bash
jupyter notebook notebooks/04_models.ipynb
```

Харьцуулах загварууд:

| Загвар | Зорилго |
|---|---|
| Шугаман регресс | Baseline, коэффициентийн тайлбар |
| Lasso / Ridge | Регулярчилал, чухал хувьсагч сонгох |
| Шийдвэрийн мод | Шугаман бус хамаарал, тайлбарлахад хялбар |
| Random Forest | Хамгийн өндөр нарийвчлал, feature importance |

**Үнэлгээний хэмжүүр**: RMSE, MAE, R² (тест дээр)

## Өгөгдлийн тухай

**Эх сурвалж**: [unegui.mn](https://www.unegui.mn/l-hdlh/l-hdlh-zarna/oron-suuts-zarna/) — Монгол улсын хамгийн том зар сурталчилгааны платформ.

**Цуглуулсан хувьсагч (22)**:

| Багана | Тайлбар |
|---|---|
| `price_mnt` | Үнэ (₮) — таамаглах гол хувьсагч |
| `area_m2` | Талбай (м²) |
| `rooms` | Өрөөний тоо |
| `floor` / `total_floors` | Хэдэн давхарт / нийт давхар |
| `year_built` | Ашиглалтад орсон он |
| `district` / `khoroo` | Дүүрэг / хороо |
| `balcony` / `garage` / `elevator` | Тагт, гараж, цахилгаан шат |
| `window_type` / `door_type` / `floor_material` | Барилгын материал |
| `construction_status` | Барилгын явц |
| `payment_terms` | Төлбөрийн нөхцөл |


- robots.txt-д тус хэсгийн scraping хориглогдоогүй
- Хүсэлт хооронд 1.5 секунд хүлээж серверт ачаалал өгөхгүй
- Хувийн мэдээлэл (утас, зарагчийн нэр) хадгалахгүй
