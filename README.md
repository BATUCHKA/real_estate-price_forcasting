# Үл хөдлөх хөрөнгийн үнийн таамаглал — өгөгдөл цуглуулалт

unegui.mn-ээс орон сууцны зарын өгөгдөл цуглуулах скрипт.

## Суулгах

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Ажиллуулах

```bash
# Туршилт: 1 хуудас (~60 зар, ~2 минут)
python scrape_unegui.py --pages 1 --out data/test.csv

# Жинхэнэ цуглуулалт: 50 хуудас (~3000 зар, ~2 цаг)
python scrape_unegui.py --pages 50 --out data/listings.csv
```

Скрипт нь өмнө татсан зарыг автоматаар алгасах тул дахин ажиллуулахад зөвхөн шинэ зар татна.

## Тохируулга

- `--pages N` — жагсаалтын хэдэн хуудас унших
- `--start N` — эхлэх хуудасны дугаар (default: 1)
- `--out path/to.csv` — гаралтын файлын зам

## Цуглуулагдах шинж чанарууд

| Багана | Тайлбар |
|---|---|
| `price_mnt` | Үнэ (төгрөг) — таамаглах гол хувьсагч |
| `area_m2` | Талбай (м²) |
| `rooms` | Өрөөний тоо (гарчгаас) |
| `floor` / `total_floors` | Хэдэн давхарт / нийт давхар |
| `year_built` | Ашиглалтад орсон он |
| `district` / `khoroo` | Дүүрэг / хороо |
| `balcony` / `garage` / `elevator` | Тагт, гараж, цахилгаан шат |
| `window_type` / `door_type` / `floor_material` | Барилгын материал |
| `construction_status` | Барилгын явц |
| `payment_terms` | Төлбөрийн нөхцөл |

## Ёс зүй

- Хүсэлт хооронд 1.5 секунд хүлээдэг — серверт ачаалал багатай
- robots.txt-д үл хөдлөхийн зар хориглогдоогүй
- Хувийн мэдээлэл (утас, зарагчийн нэр) хадгалахгүй
