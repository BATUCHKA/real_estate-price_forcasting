# Үзүүлэн слайд

Quarto revealjs дээр бэлдсэн 14 слайд.

## Render хийх

### HTML (нээж танилцуулга хийх)

```bash
cd presentation
quarto render slides.qmd
# Үр дүн: slides.html — браузераар нээж асаах
```

### PDF (хураалгахад тохиромжтой)

```bash
quarto render slides.qmd --to pdf
# Үр дүн: slides.pdf
```

PDF болгоход `chromium`/`chrome` хэрэгтэй (Quarto автоматаар татна).

## Танилцуулга үед

- **Дэлгэцийн горим**: `F` — full screen
- **Хар самбар**: `B` — black, `W` — white
- **Speaker view**: `S` — нотс бүхий дэлгэц
- **Хүснэгт**: ESC — бүх слайдын тойм
- **PDF export**: URL-д `?print-pdf` залгаж браузер хэвлэх
